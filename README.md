Modular audio visualizer made in starfall.

Create visualizers inside of /visualizers using the template
also check out the previously existing visualizers to get an idea :)

Chat Commands:
.request [url] -> Queues a song to play
.play -> alias of request
.listVisualizers -> list all visualizers in /visualizers
.load [visualizerName] -> loads the visualizer
.help -> lists all commands and their arguments
